SCREENGRAB API

API USAGE

The API allows users to save a screenshot of a web page, retrieve a list of saved screenshots and request a screenshot by id.

There is currently no authentication required to use API.

POST http://localhost/api/v1/screenshots
Add screenshot to database.

Example Request

curl --request POST "http://localhost/api/v1/screenshots" \
--header "Content-Type: application/json" \
--data "{\"source_url\": \"https://wiggle.co.uk\"}"

GET http://localhost/api/v1/screenshots
Gets a list of all submitted screenshots, in order by last created.

Example Request

curl --request GET "http://localhost/api/v1/screenshots" \
--header "Content-Type: application/json"

GET http://localhost/api/v1/screenshots/<id>
Get infro about a screenshot from database.

Example Request

curl --location --request GET "http://localhost/api/v1/screenshots/1" \
--header "Content-Type: application/json"
